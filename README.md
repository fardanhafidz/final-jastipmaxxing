# GA Jastip Route Optimizer

Aplikasi web optimasi rute jastip (jasa titip) menggunakan **Genetic Algorithm** untuk menentukan rute terbaik antar titik-titik POI (Point of Interest) di sekitar kampus ITS, dengan perhitungan jarak berbasis jalan nyata menggunakan **OSRM**.

## Teknologi

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Routing**: OSRM (Open Source Routing Machine)
- **Algoritma**: Genetic Algorithm (PCTSP)

## Cara Menjalankan

```bash
pip install -r requirements.txt
python app.py
```

Buka `http://localhost:5000` di browser.
